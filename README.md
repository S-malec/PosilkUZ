# PosiłkUZ
Projekt Android

🥗 PosiłkUZ
Aplikacja mobilna do kompleksowego zarządzania zapasami domowymi oraz odkrywania przepisów kulinarnych. Projekt łączy funkcje analityczne (skanowanie produktów) z interaktywnymi (lokalizacja, sensory).

🛠 Kluczowe Funkcjonalności
Zarządzanie kontem: Logowanie i rejestracja zabezpieczona systemem CAPTCHA.

Inteligentna kuchnia:

Skaner kodów: Wykorzystanie aparatu do szybkiej ewidencji produktów.

Shake-to-recipe: Losowanie przepisów przy użyciu akcelerometru.

Widżet: Podgląd kroków przepisu na ekranie głównym urządzenia.

Dane i Lokalizacja:

Firebase: Synchronizacja produktów i bazy przepisów w czasie rzeczywistym.

Mapy: Wyświetlanie najbliższych punktów sprzedaży w okolicy.

User Experience:

Tryb jasny i ciemny (Light/Dark Mode).

Obsługa dwóch wersji językowych.

System powiadomień oraz zintegrowane moduły reklamowe.

🏗 Architektura i Technologie
Backend: Firebase (Auth, Firestore, Cloud Messaging).

Sensory: Kamera (Barcode Scanner), Akcelerometr.

Integracje: Google Maps API, AdMob.
